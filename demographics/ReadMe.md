# Patient Service

## Overview

Manages patient demographic information.

Stores data in PostgreSQL.

## Responsibilities

- Add Patient
- Update Patient
- View Patient
- List Patients

## Technology

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL

## Validation

- Required Fields
- Phone Validation
- Date Validation
- Address Length

## Endpoints

GET /patients

GET /patients/{id}

POST /patients

PUT /patients/{id}

## Database

PostgreSQL

Patient_information
